Entendendo Comunicação Client x Server

(Voltar)

A comunicação entre cliente e servidor é um conceito fundamental na construção de aplicações web.

Essa troca de informações permite que os usuários interajam com os servidores e recebam as respostas necessárias para suas solicitações.

Introdução ao HTML na prática

(Voltar)

O HTML* (HyperText Markup Language) é a linguagem fundamental para a construção de páginas web.

É uma linguagem de marcação que permite definir a estrutura e o conteúdo de um documento, como textos, imagens, links e outros elementos.

Como entregar seu desafio de projeto
(Voltar)

O GitHub é uma plataforma amplamente utilizada para o controle de versão e colaboração em projetos de desenvolvimento de software.

É também um excelente meio para entregar e compartilhar seus desafios de projeto de forma organizada e acessível.

Passo a Passo

  1. Crie um repositório
  2. Estruture seu projeto
  3. Faça o upload dos arquivos
  4. Adicione um README
  5. Documente seu código
  6. Commits e Push
  7. Branches (ramificações)
  8. Pull request (solicitação de pull)
  9. Compartilhe o link
